# Hype Client

Hype Client is a private macOS-focused Minecraft Java Edition 1.8.9 client and launcher developed for personal use and testing.

The project focuses on:

* macOS fullscreen stability improvements
* CPU and rendering optimisation
* HUD customisation
* visual and quality-of-life features
* launcher and profile management

Hype Client uses official Microsoft OAuth authentication only.

The project does not support cracked accounts, account bypassing, credential collection, or redistribution of Minecraft assets.

Current implemented features include:

* Lighting / Fullbright
* Freelook
* 1.7 Animations
* Zoom
* Crosshair Customisation
* Motion Blur
* Toggle Sprint / Sneak
* FPS Counter
* Keystrokes
* Potion Status
* Armour Status
* Item Durability
* Ping Display
* Coordinates
* Direction HUD
* Memory Usage
* Clock
* Bossbar and Scoreboard Customisation

This project is currently non-commercial and privately developed.
